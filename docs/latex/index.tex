This repository contains my implementation of feature-\/based EKF SLAM with unsupervised learning. I implemented all the packages from Scratch using ROS in C++.~\newline
~\newline
 Please visit \href{https://yaelbenshalom.github.io/EKF_SLAM/index.html}{\texttt{ my website}} for more information about this project.~\newline
 For further information about the packages, classes, and methods used in this project, please visit \href{https://yaelbenshalom.github.io/Turtlebot3-SLAM-from-scratch/html/index.html}{\texttt{ package documentation}}.\hypertarget{index_autotoc_md21}{}\doxysection{Package List}\label{index_autotoc_md21}
This repository consists of several ROS packages\+:


\begin{DoxyItemize}
\item \href{https://github.com/YaelBenShalom/Turtlebot3-SLAM-from-scratch/tree/master/nuturtle_description}{\texttt{ {\bfseries{nuturtle\+\_\+description}}}} -\/ A package that adapts the turtlebot3\+\_\+burger, a differential drive robot, for our needs.
\item \href{https://github.com/YaelBenShalom/Turtlebot3-SLAM-from-scratch/tree/master/rigid2d}{\texttt{ {\bfseries{rigid2d}}}} -\/ A package that handles transformations in SE(2).
\item \href{https://github.com/YaelBenShalom/Turtlebot3-SLAM-from-scratch/tree/master/trect}{\texttt{ {\bfseries{trect}}}} -\/ A package that causes the turtlesim turtle to follow a rectangle path.
\item \href{https://github.com/YaelBenShalom/Turtlebot3-SLAM-from-scratch/tree/master/nuturtle_robot}{\texttt{ {\bfseries{nuturtle\+\_\+robot}}}} -\/ A package that stores the code that interacts with the turtlebot hardware.
\item \href{https://github.com/YaelBenShalom/Turtlebot3-SLAM-from-scratch/tree/master/nurtlesim}{\texttt{ {\bfseries{nurtlesim}}}} -\/ A package that simulates the robot kinematics and a sensor that detects the relative x, y positions of landmarks and a landmark id.
\item \href{https://github.com/YaelBenShalom/Turtlebot3-SLAM-from-scratch/tree/master/nuslam}{\texttt{ {\bfseries{nuslam}}}} -\/ A package contains the implementation Feature-\/\+Based Kalman Filter SLAM.
\end{DoxyItemize}

This repository uses external package\+:


\begin{DoxyItemize}
\item {\bfseries{nuturtlebot}} -\/ Additional package to help us work with the lower-\/level hardware on the turtlebot. In order to run this project, please download the {\ttfamily turtle.\+rosinstall} file.
