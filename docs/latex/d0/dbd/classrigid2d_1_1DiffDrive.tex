\doxysubsection{Detailed Description}
The class models the kinematics of a differential drive robot with a given wheel base and wheel radius. The class\+: 


\begin{DoxyEnumerate}
\item Track the configuration of a differential-\/drive robot
\item Convert a desired twist to the equivalent wheel velocities required to achieve that twist
\item Update the configuration of the robot, given updated wheel angles (assuming constant wheel velocity in-\/between updates) 
\end{DoxyEnumerate}
